# My first OS

My first OS written in Rust

## Operating Environments

* 64bit UEFI 2.X + ACPI 2.X
* x64 processor with up to 64 cores
* XX MB of system memory
* 800x600 pixels screen
* PS/2 keyboard and mouse
* XHCI (in the future)

## Requirements to Build

* Rust nightly
* qemu + ovmf (optional)

### how to build

```
$ make
```

### how to run

```
$ make run
```

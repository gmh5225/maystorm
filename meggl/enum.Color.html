Enum meggl::Color

source · 
pub enum Color {
    Transparent,
    Indexed(IndexedColor),
    Argb32(ARGB8888),
}
Expand description

A type that represents a generic color.

The Color type is convertible to the PackedColor type and each other, with some exceptions.
